1. Write in detail about Data Types in Java

1. Introduction

Java is a strongly typed, object-oriented, and platform-independent programming language. To handle different kinds of data efficiently and securely, Java provides a well-defined system of data types.

2. Definition

A data type in Java specifies:

  • Type of data a variable can store
  • Amount of memory allocated
  • Range of values
  • Operations permitted

3. Importance of Data Types

  • Efficient memory management
  • Prevents type mismatch errors
  • Improves program reliability
  • Ensures platform independence
  • Compile-time error checking

4. Classification of Data Types in Java

  1. Primitive Data Types
  2. Non-Primitive (Reference) Data Types

Program: Integer Data Types


class IntegerDataTypes {
    public static void main(String[] args) {
        int age = 18;
        long population = 1400000000L;

        System.out.println("Age: " + age);
        System.out.println("Population: " + population);
    }
}

Write in detail about Operators in Java

1. Introduction

Operators are symbols used to perform operations on operands. Java provides a rich set of operators for computation, decision making, and bit manipulation.

2. Classification of Operators

  1. Arithmetic
  2. Unary
  3. Relational
  4. Logical
  5. Assignment
  6. Bitwise
  7. Shift
  8. Ternary

3. Arithmetic Operators

OperatorOperation
+Addition
-Subtraction
*Multiplication
/Division
%Modulus

Example


int a = 10;
int b = 3;
System.out.println(a + b);
System.out.println(a % b);
